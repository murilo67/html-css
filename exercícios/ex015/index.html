Usando links...

Links externos

Sobrepondo o seu site

Podemos acessar links externos graças à uma tag chamada "a" para colocar um link, o ue se pode colocar também a um trecho como fazemos na tag "p":
Exemplo: Repositório Público

Mantendo o seu site

Támbem podemos acessar links externos, mantendo o seu site aberto no seu navegador, para isso, podemos utilizar a tag target e rel ao lado do seu link
Exemplo: Youtube

Links internos

links da propria página ou site, para isso podemos fazer os seguintes passos:

  1. Criando um arquivo html
  2. Chama a tag a
  3. Coloca o nome no arquivo ou site

Exemplo:
Nossa segunda página
Uma forma de voltar do link para o link antigo é usar a tag a e dentro dela colocar o arquivo anterior em target você coloca "_self"
Bom falamos que dentro da tag a existe o rel prev (anterior) e o rel (next), que faz a relação com um site e outra da mesma marca
