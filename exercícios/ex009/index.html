Formatações adicionais

Texto marcado

Podemos criar um texto marcado usando a tag MARK
Agora para mudar a cor da tag MARK é dentro da tag depois da tag mark colocar style e colocar a cor (Semântica)

Texto grande e pequeno

Podemos criar quanto um texto grande, com a tag big (não é utilizada), quanto um texto pequeno com a tag small

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, com a tag del, mas não considerado

Texto inserido

Podemos marcar um texto como inserido, com a tag ins, para dar uma ênfase e indicar que ele foi adicionado depois (semântica)
Também pode ser utilizado a tag u (não semântica), que é a mesma coisa da tag ins

Texto sobrescrito

Para inserir coisas do tipo: x20 = 1, podemos utilizar a tag sup (Para cima)
Para inserir coisas do tipo: H20, com a tag sub (Para baixo)
